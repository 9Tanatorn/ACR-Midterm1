Part 1 — การหมุนเวียนของอากาศ

1.1 ทำไมโลกได้รับความร้อนไม่เท่ากัน?

🔴 โลกเป็นทรงกลม → มุมตกกระทบแตกต่าง

เนื่องจากโลกมีรูปทรงกลม รังสีจากดวงอาทิตย์จึงตกกระทบพื้นผิวโลกในมุมที่แตกต่างกัน บริเวณเส้นศูนย์สูตรแสงตกเกือบตั้งฉาก (มุมตกกระทบสูง) ทำให้พลังงานกระจุกตัวในพื้นที่เล็ก → ร้อนมาก ส่วนบริเวณขั้วโลกแสงตกเฉียง พลังงานกระจายในพื้นที่กว้าง → เย็น

💡 ความแตกต่างของอุณหภูมิระหว่างเส้นศูนย์สูตรและขั้วโลก คือ "ต้นกำเนิด" ของลมและการหมุนเวียนอากาศทั้งหมด

🔴 แกนโลกเอียง 23.5° → ฤดูกาล

แกนโลกเอียงทำมุม 23.5° กับแนวตั้งฉากของระนาบวงโคจร ส่งผลให้ในช่วงเวลาต่างกันของปี ซีกโลกเหนือและซีกโลกใต้ได้รับแสงอาทิตย์ไม่เท่ากัน → เกิดฤดูกาล

  • Summer Solstice (21 มิ.ย.): ซีกเหนือเอียงเข้าหาดวงอาทิตย์ → ฤดูร้อนซีกเหนือ
  • Winter Solstice (21 ธ.ค.): ซีกใต้เอียงเข้าหาดวงอาทิตย์ → ฤดูหนาวซีกเหนือ
  • Equinox (21 มี.ค., 23 ก.ย.): ทั้งสองซีกได้รับแสงเท่ากัน → กลางวัน = กลางคืน

🔴 โลกหมุนรอบตัวเอง → Coriolis Effect

การหมุนรอบตัวเองของโลก (ตะวันตก→ตะวันออก) ทำให้วัตถุที่เคลื่อนที่บนพื้นผิวโลก "ถูกเบน" ออกจากแนวเส้นตรง เรียกว่าแรงคอริออลิส (Coriolis Force)

  • ซีกโลกเหนือ: ลมเบนไปทางขวาของทิศทางการเคลื่อนที่
  • ซีกโลกใต้: ลมเบนไปทางซ้ายของทิศทางการเคลื่อนที่
  • บริเวณเส้นศูนย์สูตร: Coriolis effect = 0 (ไม่มีการเบน)

1.2 Atmospheric Circulation — 3 Cells

ความแตกต่างของอุณหภูมิระหว่างเส้นศูนย์สูตรกับขั้วโลก ร่วมกับ Coriolis Effect ทำให้การไหลเวียนของอากาศแบ่งออกเป็น 3 วงจร (Cells) ในแต่ละซีกโลก:

Cellละติจูดลมผิวพื้นลักษณะอากาศ
Hadley0°–30°ลมค้า (NE/SE Trade Winds)ITCZ ฝนชุก, 30° แห้งแล้ง (ทะเลทราย)
Ferrel30°–60°ลมตะวันตก (Westerlies)อากาศแปรปรวน, พายุบ่อย
Polar60°–90°ลมขั้วโลก (Polar Easterlies)แห้ง เย็น, Polar front

1.3 ลมบก — ลมทะเล (Land & Sea Breeze)

เกิดจากความแตกต่างของความจุความร้อนจำเพาะระหว่างดินกับน้ำ น้ำทะเลมีค่า specific heat capacity สูงกว่าพื้นดิน → ดินร้อนเร็ว-เย็นเร็ว ส่วนน้ำร้อนช้า-เย็นช้า

💡 จำง่าย: กลางวัน = ดินร้อน → อากาศเหนือดินลอยขึ้น → ลมจากทะเลพัดเข้าฝั่ง (ลมทะเล) | กลางคืน = ตรงข้ามเลย

1.4 ลมมรสุม (Monsoon)

ลมมรสุมเป็น "ลมบก-ลมทะเล ขนาดใหญ่ระดับทวีป" ที่เปลี่ยนทิศตามฤดูกาล เกิดจากความแตกต่างของอุณหภูมิระหว่างแผ่นดินขนาดใหญ่กับมหาสมุทร

🌧️ มรสุมตะวันตกเฉียงใต้ (พ.ค.–ต.ค.)

แผ่นดินเอเชียร้อนจัด → ดูดลมจากมหาสมุทรอินเดีย → พัดพาความชื้นมาก → ฤดูฝนของไทย

❄️ มรสุมตะวันออกเฉียงเหนือ (พ.ย.–ก.พ.)

แผ่นดินเอเชียเย็นจัด (แรงดันสูง) → ลมพัดจากจีน/ไซบีเรียลงมา → แห้ง เย็น → ฤดูหนาวของไทย

🇹🇭 ประเทศไทยได้รับอิทธิพลจากทั้งสองมรสุมเพราะตั้งอยู่ในเขตมรสุมเอเชีย (Asian Monsoon Region)

1.5 El Niño / La Niña / ENSO

ENSO (El Niño–Southern Oscillation) คือปรากฏการณ์ที่อุณหภูมิผิวน้ำทะเลในมหาสมุทรแปซิฟิกเขตศูนย์สูตรเปลี่ยนแปลงผิดปกติ ส่งผลต่อ Walker Circulation และภูมิอากาศทั่วโลก

สภาวะลมค้าอุณหภูมิน้ำทะเลฝั่ง EUpwellingผลต่อไทย/เอเชีย
ปกติแรงปกติเย็น (ปกติ)ปกติฝนตามฤดูกาลปกติ
El Niñoอ่อนลง/กลับทิศอุ่นผิดปกติ (+0.5°C↑)ลดลง/หยุดแล้งกว่าปกติ ฝนน้อย
La Niñaแรงขึ้นเย็นผิดปกติ (−0.5°C↓)แรงมากขึ้นฝนมากกว่าปกติ น้ำท่วม

1.6 พายุหมุนเขตร้อนและพายุฝนฟ้าคะนอง

🌀 พายุหมุนเขตร้อน (Tropical Cyclone)

ก่อตัวเหนือมหาสมุทรเขตร้อนที่อุณหภูมิผิวน้ำ ≥ 26.5°C อากาศชื้นลอยตัวขึ้นเป็นวงหมุน (ซีกเหนือหมุนทวนเข็มนาฬิกา เพราะ Coriolis) ปล่อยพลังงานมหาศาลจากความร้อนแฝงของการควบแน่น

ระดับความเร็วลมชื่อเรียก (ไทย)ชื่อเรียก (สากล)
1< 63 km/hพายุดีเปรสชันTropical Depression
263–117 km/hพายุโซนร้อนTropical Storm
3118–220 km/hไต้ฝุ่นTyphoon / Hurricane / Cyclone
4+> 220 km/hซูเปอร์ไต้ฝุ่นSuper Typhoon (Cat 5)
🌏 ชื่อเรียกต่างกันตามภูมิภาค: Typhoon (แปซิฟิกตะวันตก), Hurricane (แอตแลนติก), Cyclone (มหาสมุทรอินเดีย)

⛈️ พายุฝนฟ้าคะนอง (Thunderstorm)

เกิดจากอากาศร้อนชื้นลอยตัวขึ้นอย่างรวดเร็ว (Convection) → เกิดเมฆคิวมูโลนิมบัส (Cb) → ฟ้าแลบ ฟ้าร้อง ฝนหนัก ลมกระโชก อาจมีลูกเห็บ

🌪️ ทอร์นาโด (Tornado)

ลมหมุนรุนแรงขนาดเล็กที่ยื่นจากเมฆ Cb ลงถึงพื้นดิน มีเส้นผ่านศูนย์กลาง 100–600 m ลมความเร็วถึง 500 km/h พบมากในสหรัฐอเมริกา (Tornado Alley)

1.7 What If? — ถ้าโลกเปลี่ยนไป

ถ้าโลกหยุดหมุน?

• ไม่มี Coriolis Effect → ลมพัดตรงจากขั้วโลกสู่ศูนย์สูตร
• ไม่มีกลางวัน-กลางคืน 12 ชม. → ด้านหนึ่งร้อนจัดตลอด อีกด้านเย็นจัดตลอด
• ไม่เกิด 3 Cells → ไม่มีลมค้า ไม่มี Westerlies
• สนามแม่เหล็กโลกหายไป → รังสีอวกาศทำลายสิ่งมีชีวิต

ถ้าโลกไม่เอียง?

• ไม่มีฤดูกาล → อุณหภูมิคงที่ตลอดปี
• ไม่มีลมมรสุม → เพราะไม่มีการเปลี่ยนแปลงตามฤดู
• บริเวณเส้นศูนย์สูตรร้อนตลอด ขั้วโลกเย็นตลอด
• ระบบนิเวศเปลี่ยนแปลงรุนแรง การเกษตรกระทบ

ถ้าแสงอาทิตย์ไม่ตกตรง (มุมเฉียงทุกที่)?

• พลังงานกระจายบนพื้นที่กว้าง → โลกทั้งใบเย็นลง
• ไม่มีเขตร้อน ไม่มี ITCZ
• 3 Cells อ่อนลงมาก → ลมอ่อน ฝนน้อย
• มหาสมุทรแข็งตัวจากขั้วโลกลงมา → ยุคน้ำแข็ง

ถ้าโลกไม่เป็นทรงกลม (แบน)?

• แสงตกกระทบทุกที่เท่ากัน → อุณหภูมิเท่ากันทั่วโลก
• ไม่มี gradient อุณหภูมิ → ไม่เกิดลม
• ไม่มีความกดอากาศแตกต่าง → สภาพอากาศนิ่ง
• แรงโน้มถ่วงไม่สม่ำเสมอ → น้ำกองกลาง

Part 2 — ธรณีพิบัติภัย

2.1 ภูเขาไฟระเบิด

เกิดจากการแทรกดันของแมกมาขึ้นมาบนผิวโลกตามช่องว่างหรือรอยแตกของเปลือกโลก ตำแหน่งส่วนใหญ่สัมพันธ์กับแนวรอยต่อของแผ่นธรณี

ประเภทของภูเขาไฟ

💡 ภูเขาไฟที่เกิดจาก จุดร้อน (Hot Spot) เช่น Hawaii ไม่ได้อยู่บนแนวรอยต่อแผ่นธรณี แต่เกิดจากแมกมาแทรกดันจากชั้น mantle ลึก

2.2 แผ่นดินไหว

เกิดจากแผ่นธรณีเคลื่อนที่ → เสียดสี/ชนกัน → หินสะสมพลังงานจนแตกหัก → ปลดปล่อยพลังงานเป็นคลื่นไหวสะเทือน (Seismic Waves)

ศูนย์เกิด vs จุดเหนือศูนย์เกิด

  • ศูนย์เกิดแผ่นดินไหว (Focus/Hypocenter) = ตำแหน่งใต้ผิวโลกที่หินแตกหัก ปลดปล่อยพลังงาน
  • จุดเหนือศูนย์เกิด (Epicenter) = จุดบนผิวโลกที่อยู่ตรงเหนือศูนย์เกิด → ได้รับผลกระทบมากที่สุด

ขนาด (Magnitude) vs ความรุนแรง (Intensity)

⚠️ หัวข้อนี้ออกข้อสอบบ่อยมาก — ต้องแยกให้ชัดว่าวัดอะไร ด้วยเครื่องมืออะไร
ขนาด (Magnitude)ความรุนแรง (Intensity)
วัดอะไรปริมาณพลังงานที่ปลดปล่อยผลกระทบ/ความเสียหายที่เกิดขึ้น
วัดจากอะไรแอมพลิจูดคลื่นจากเครื่อง Seismographความรู้สึกผู้คน + ความเสียหายสิ่งก่อสร้าง
มาตราที่ใช้มาตราริกเตอร์ / มาตราขนาดโมเมนต์ (Mw)มาตราเมอร์คัลลีปรับปรุงใหม่ (MMI)
ค่าคงที่?มีค่าเดียวต่อเหตุการณ์แตกต่างตามระยะห่างจาก Epicenter
ตัวอย่างแผ่นดินไหวขนาด 7.0MMI XII ใกล้ Epicenter, MMI III ไกลออกไป
📌 ทำไมมาตราเมอร์คัลลี (MMI) นิยมใช้? เพราะประเมินจากความเสียหายที่เกิดขึ้นจริง ไม่ต้องใช้เครื่องมือราคาแพง สามารถสำรวจย้อนหลังได้ และบอกได้ว่าแต่ละพื้นที่ได้รับผลกระทบมากน้อยเพียงใด ทำเป็น "แผนที่ความเสียหาย" ได้ ซึ่งมาตราริกเตอร์ทำไม่ได้ (ให้ค่าเดียว)

การปฏิบัติตนเมื่อเกิดแผ่นดินไหว

ก่อนเกิด

• เตรียมกระเป๋าฉุกเฉิน
• จัดวางเฟอร์นิเจอร์ยึดกับผนัง
• ฝึกซ้อมแผนอพยพ
• รู้ตำแหน่งวาล์วน้ำ-ไฟฟ้า

ระหว่างเกิด

• ในอาคาร: หมอบ-คลุม-เกาะ (Drop-Cover-Hold) ใต้โต๊ะแข็ง
• กลางแจ้ง: ออกห่างอาคาร สายไฟ เสาไฟ
• ในรถ: จอดข้างทาง ไม่จอดใต้สะพาน
• ห้ามวิ่ง! ห้ามใช้ลิฟต์!

หลังเกิด

• ระวัง aftershock (แผ่นดินไหวตาม)
• ตรวจสอบอาคารก่อนเข้า
• ปิดแก๊ส-ไฟฟ้าถ้าสงสัยรั่ว
• ติดตามข่าวจากหน่วยงานราชการ

2.3 สึนามิ (Tsunami)

เกิดจากการแทนที่มวลน้ำอย่างเฉียบพลัน สาเหตุหลักคือแผ่นดินไหวใต้ทะเลที่ทำให้พื้นมหาสมุทรเลื่อนตัวในแนวดิ่ง (บริเวณแนวมุดตัวของแผ่นธรณี)

ลักษณะเฉพาะของสึนามิ

ลักษณะในทะเลลึกเข้าสู่ชายฝั่ง (น้ำตื้น)
ความเร็วสูงมาก 500–800 km/hลดลง 30–60 km/h
ความยาวคลื่น100–200 kmสั้นลง
ความสูงคลื่นน้อยมาก 30–60 cmสูงขึ้นมาก 10–30+ m
⚠️ สัญญาณเตือนธรรมชาติ: น้ำทะเลลดระดับลงอย่างรวดเร็วผิดปกติ → รีบอพยพขึ้นที่สูง! ระบบเตือนภัย DART (Deep Ocean Assessment and Reporting of Tsunami) ตรวจวัดแรงดันน้ำใต้มหาสมุทร

ข้อ 1: แผ่นดินไหว "ขนาด" กับ "ความรุนแรง" วัดต่างกันอย่างไร?

อธิบายว่าขนาด (Magnitude) และความรุนแรง (Intensity) ของแผ่นดินไหวแตกต่างกันอย่างไร ใช้เครื่องมือหรือวิธีการใดในการวัด

แนวคำตอบ

ขนาด (Magnitude) คือค่าที่บ่งบอกปริมาณพลังงานที่ถูกปลดปล่อยจากศูนย์เกิดแผ่นดินไหว คำนวณจากค่าแอมพลิจูดของคลื่นไหวสะเทือนที่ตรวจวัดด้วยเครื่อง Seismograph แผ่นดินไหวแต่ละครั้งมีค่าขนาดเพียงค่าเดียว รายงานด้วยมาตราริกเตอร์หรือมาตราขนาดโมเมนต์ (Mw)

ความรุนแรง (Intensity) คือการประเมินผลกระทบจากแผ่นดินไหว ได้แก่ ความรู้สึกของผู้คน ความเสียหายของสิ่งก่อสร้าง และการเปลี่ยนแปลงของภูมิประเทศ ใช้มาตราเมอร์คัลลีปรับปรุงใหม่ (MMI) มีค่าแตกต่างกันตามระยะห่างจากจุดเหนือศูนย์เกิดแผ่นดินไหว

Keyword ที่ต้องมี
  • พลังงาน / แอมพลิจูด / Seismograph / ริกเตอร์ / Mw
  • ผลกระทบ / ความเสียหาย / เมอร์คัลลี (MMI)
  • ค่าเดียว vs แตกต่างตามระยะทาง

ข้อ 2: มาตราวัดความรุนแรงที่นิยมใช้คืออะไร ทำไมจึงนิยมใช้?

ระบุชื่อมาตราและอธิบายเหตุผลที่มาตรานี้ได้รับความนิยม

แนวคำตอบ

มาตราที่นิยมใช้วัดความรุนแรงคือ มาตราเมอร์คัลลีปรับปรุงใหม่ (Modified Mercalli Intensity - MMI) แบ่ง 12 ระดับ (I–XII)

เหตุผลที่นิยมใช้: (1) ประเมินจากความเสียหายจริงที่เกิดขึ้น ไม่จำเป็นต้องใช้เครื่องมือราคาแพง (2) สามารถสำรวจได้ภายหลังจากเหตุการณ์ (3) สร้างแผนที่แสดงระดับความเสียหายในแต่ละพื้นที่ได้ (ShakeMap) ซึ่งมีประโยชน์ต่อการบรรเทาสาธารณภัย (4) ใช้ได้กับแผ่นดินไหวในอดีตที่ไม่มีเครื่องมือบันทึก

ข้อ 3: อธิบายวิธีเอาตัวรอดจากแผ่นดินไหว (ก่อน ระหว่าง หลัง)

แนวคำตอบ

ก่อนเกิด: เตรียมกระเป๋าฉุกเฉิน ยึดเฟอร์นิเจอร์หนัก ฝึกซ้อมแผนอพยพ รู้ตำแหน่งวาล์วน้ำ-ไฟ-แก๊ส

ระหว่างเกิด: ในอาคาร → หมอบ-คลุม-เกาะ (Drop-Cover-Hold On) ใต้โต๊ะแข็งแรง ห้ามใช้ลิฟต์ กลางแจ้ง → ออกห่างอาคาร สายไฟ ในรถ → จอดข้างทาง

หลังเกิด: ระวัง aftershock ตรวจสอบความเสียหายอาคาร ปิดแก๊ส-ไฟฟ้า ติดตามข่าวจากกรมอุตุฯ หากอยู่ชายฝั่ง + รู้สึกแผ่นดินไหวหรือน้ำทะเลลดผิดปกติ → อพยพขึ้นที่สูงทันที (อาจเกิดสึนามิ)

ข้อ 4: อธิบายการหมุนเวียนอากาศ 3 Cells

ระบุชื่อ ตำแหน่งละติจูด และทิศทางลมผิวพื้นของแต่ละ Cell

แนวคำตอบ

การหมุนเวียนอากาศในแต่ละซีกโลกแบ่งเป็น 3 วงจร (Cell) เนื่องจากความแตกต่างของอุณหภูมิระหว่างเส้นศูนย์สูตรกับขั้วโลก ร่วมกับ Coriolis Effect จากการหมุนรอบตัวเองของโลก:

  • Hadley Cell (0°–30°): อากาศร้อนที่เส้นศูนย์สูตรลอยสูงขึ้น → ไหลไปทางขั้วโลก → จมลงที่ 30° → ไหลกลับที่ผิวพื้นเป็น "ลมค้า" (NE Trade Winds ซีกเหนือ, SE Trade Winds ซีกใต้)
  • Ferrel Cell (30°–60°): อากาศที่จมลงที่ 30° บางส่วนไหลไปทาง 60° → ถูก Coriolis เบน → เกิด "ลมตะวันตก" (Westerlies) อากาศแปรปรวน
  • Polar Cell (60°–90°): อากาศเย็นจากขั้วโลกจมลง → ไหลไปทาง 60° → เกิด "ลมขั้วโลก" (Polar Easterlies) เย็นและแห้ง

ข้อ 5: อธิบายลมมรสุมที่มีอิทธิพลต่อประเทศไทย

แนวคำตอบ

ประเทศไทยได้รับอิทธิพลจากลมมรสุม 2 ฤดู ซึ่งเป็น "ลมบก-ลมทะเลขนาดทวีป":

  • มรสุมตะวันตกเฉียงใต้ (พ.ค.–ต.ค.): แผ่นดินเอเชียร้อน (ความกดอากาศต่ำ) → ดูดลมจากมหาสมุทรอินเดีย → พาความชื้นมาก → ทำให้เกิดฤดูฝนในไทย
  • มรสุมตะวันออกเฉียงเหนือ (พ.ย.–ก.พ.): แผ่นดินเอเชียเย็น (ความกดอากาศสูง) → ลมหนาวจากไซบีเรีย/จีนพัดลงมา → แห้งและเย็น → ทำให้เกิดฤดูหนาวในไทย (ฝนตกชายฝั่งตะวันออกเพราะรับความชื้นจากอ่าวไทย)
Keyword
  • ความกดอากาศ สูง/ต่ำ, ทิศ, ความชื้น, ฤดูกาล

ข้อ 6: ถ้าโลกหยุดหมุน ไม่เอียง จะเกิดอะไรขึ้นกับระบบอากาศ?

แนวคำตอบ

ถ้าหยุดหมุน: (1) ไม่มี Coriolis Effect → ลมพัดตรงจากความกดอากาศสูงไปต่ำ ไม่มีการเบน → ไม่เกิด 3 Cells (2) กลางวัน-กลางคืนยาวนานมาก (~6 เดือน) → ด้านรับแสงร้อนจัด ด้านมืดเย็นจัด (3) ความแตกต่างอุณหภูมิรุนแรง → ลมแรงจัดจากด้านเย็นไปด้านร้อน (4) สนามแม่เหล็กโลกหายไป → รังสีคอสมิกทำลายสิ่งมีชีวิต

ถ้าไม่เอียง: (1) ไม่มีฤดูกาล → อุณหภูมิคงที่ตลอดปีในแต่ละละติจูด (2) ไม่มีลมมรสุม → เพราะไม่มีการเปลี่ยนแปลงตามฤดู (3) เส้นศูนย์สูตรร้อนจัดถาวร ขั้วโลกเย็นตลอด (4) ป่าฝนและระบบนิเวศเปลี่ยนแปลงรุนแรง

ข้อ 7: ถ้าแสงอาทิตย์ไม่ตกกระทบพื้นผิวโลกในแนวตั้งฉากเลย จะเกิดอะไรขึ้น?

แนวคำตอบ

(1) พลังงานจากดวงอาทิตย์กระจายบนพื้นที่กว้าง → ทุกบริเวณได้รับพลังงานน้อยลง → อุณหภูมิเฉลี่ยของโลกลดลง

(2) ไม่มีเขตร้อน (Tropics) เพราะไม่มีจุดที่รับแสงตรง → ไม่มี ITCZ (เขตบรรจบลมค้า) → ฝนลดลงทั่วโลก

(3) การหมุนเวียนอากาศ 3 Cells อ่อนลงมาก เพราะ gradient อุณหภูมิระหว่างเส้นศูนย์สูตรกับขั้วโลกลดลง → ลมอ่อน พายุน้อยลง

(4) มหาสมุทรเริ่มแข็งตัวจากขั้วโลกลงมา → เข้าสู่สภาวะคล้ายยุคน้ำแข็ง สิ่งมีชีวิตต้องปรับตัวหรือสูญพันธุ์
